## Getting started

<Card title="Install threatcl" icon="star">
  <Steps>

1. Install with homebrew:

   ```bash
   brew install threatcl/repo/threatcl
   ```

2. Or, install with go:

   ```bash
   go install github.com/threatcl/threatcl/cmd/threatcl@latest
   ```

3. Or, run with docker:

   ```bash
   docker run --rm -it ghcr.io/threatcl/threatcl
   ```

4. Or, integrate with GitHub Actions:

	<LinkCard title="github.com/threatcl/threatcl-action" icon="github" description="Learn more here" href="https://github.com/threatcl/threatcl-action" />
  </Steps>
